# AGENTS.md

## 项目说明

Daily Todo，一个简单的每日任务清单，适合大一计算机学生练习前端开发。

## 技术栈

- 原生 HTML
- 原生 CSS
- 原生 JavaScript
- 不使用任何框架或构建工具

## 修改规则

### 必须遵守

1. **不要直接改 main**：所有修改必须新建分支，通过 Pull Request 合并。
2. **保持原生技术栈**：禁止引入 React、Vue、Vite、Tailwind、Bootstrap 等框架。
3. **不要添加依赖**：不创建 package.json，不安装 node_modules。
4. **保持简单**：代码清晰，适合初学者理解。
5. **小步提交**：每次只做一个小改动，不要一次性修改大量代码。

### 修改后检查

- [ ] 页面能直接用浏览器打开
- [ ] 没有引入新依赖
- [ ] 手机端布局正常
- [ ] 没有改动无关文件

## 工作流程

1. 从 `main` 创建新分支
2. 进行小范围修改
3. 提交并推送
4. 使用 `gh pr create` 创建 Pull Request
